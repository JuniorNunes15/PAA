Questão 2

Passo base:
Verificar se a afirmação é verdadeira para o menor valor possível de k e j, que é k = j = 2. Nesse caso, temos que 2 + 2 = 4 é um número par. Portanto, a afirmação é verdadeira para k = j = 2.

Passo de indução:
Suponha que a afirmação é verdadeira para dois números pares k e j, isto é, k + j é um número par. Precisamos provar que a afirmação também é verdadeira para k + 2 e j + 2, isto é, que (k + 2) + (j + 2) é um número par.

Podemos simplificar a expressão acima para obter:
(k + 2) + (j + 2) = k + j + 4
Observe que k + j é um número par, já que ambos são números pares, e que 4 é um número par. A soma de dois números pares é sempre um número par e, portanto, k + j + 4 é um número par.
